# Arabic Healthcare Dataset Preprocessing 

## Overview

This repository contains my contribution to a graduation project focused on building an AI-based medical assistant chatbot. My role involved designing and implementing the preprocessing and data cleaning pipeline for an Arabic medical Q&A dataset.

## My Contribution
-Cleaned and preprocessed raw Arabic medical text data.

-Normalized text (handling variations in Arabic writing).

-Tokenized and processed text using Stanza.

-Structured inconsistent medical categories into standardized labels.

-Prepared the dataset for downstream machine learning models.

## Techniques Used
Arabic NLP preprocessing.

Text normalization (handling diacritics, different letter forms, and inconsistencies).

Tokenization using Stanza.

Stopword removal.

Text cleaning (removing noise, punctuation, and irrelevant characters).

Handling Arabic morphological variations.

Data labeling and category standardization.

Dataset structuring for supervised learning.

Basic exploratory data analysis (EDA).

Data validation and consistency checks.
